As the name suggest `@sheepdog/svelte` is meant to be used in a `svelte` project.

To install it run the following command with your preferred package manager.

<Tabs>
	<TabItem label="npm">
		<Code lang="bash" code="npm install @sheepdog/svelte" />
	</TabItem>
	<TabItem label="yarn">
		<Code lang="bash" code="yarn add @sheepdog/svelte" />
	</TabItem>
	<TabItem label="pnpm">
		<Code lang="bash" code="pnpm install @sheepdog/svelte" />
	</TabItem>
	<TabItem label="bun">
		<Code lang="bash" code="bun add @sheepdog/svelte" />
	</TabItem>
</Tabs>

## Setup the Async Transform

<Aside>
	This is an optional part of the configuration, if you want to know more about the Async Transform
	to decide if you want to include it in your project check out the [the Async Transform
	guide](/guides/async-transform)
</Aside>

The Async transform is a Vite plugin so to set it up you need to modify your `vite.config.{js|ts}`

```diff lang="ts" title="vite.config.ts"
import { sveltekit } from '@sveltejs/kit/vite';
+import { sheepdogTransform } from '@sheepdog/svelte/lib/vite';
import { defineConfig } from 'vitest/config';

export default defineConfig(({ mode }) => ({
	plugins: [
		sveltekit(),
+		sheepdogTransform(),
	],
	test: {
		include: ['src/**/*.{test,spec}.{js,ts}'],
	},
}));
```

And that's it! You can now use async tasks with proper mid-run cancellation!
